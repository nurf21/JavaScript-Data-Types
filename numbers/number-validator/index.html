Number Input Validator

This utility will prompt you to enter a numeric value until you provide a valid number or cancel the operation.

How it works:

  • Click the "Enter Number" button to start
  • Enter any value in the prompt dialog
  • The system will validate your input
  • If the value is not a valid number, you'll be prompted again
  • You can cancel at any time by clicking "Cancel" or entering an empty string
